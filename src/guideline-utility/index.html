Border radius can be used in conjunction with a cap-top class to add an image to the top of a call-to-action with two rounded corners.

Additionally, any image can become round by adding a round class onto it.

Note that by default, images will break out of their containers. If you wish to restrain an image to to more than 100% of its container, a class of img-fluid should be added to it.

Borders

Borders are sparingly used but are sometimes applied around icons. The border-thin class can be used for this purpose.

Vertical spacing

Vertical spacing is achieved using four different classes:

  • gutter-large adds a large space under elements. It is mostly used on the home page.
  • gutter-medium is a useful class for adding white space between blocks of discrete content on the page
  • gutter-small is sometimes useful within blocks, for adding a small amount of whitespace
  • flush removes any pre-existing margin bottom on an element

Additionally, images can be persuaded to sit closer together using the overlap-small class. This pulls the following image up with a negative margin, allowing for some off-grid layouts.

Text alignment

Text can be aligned with a few different classes:

  • text-left aligns text to the left (note that most text is already aligned this way)
  • text-center aligns text to the centre
  • text-right aligns text to the right

Text adjustment

Text can be adjusted in a few ways:

  • text-uppercase converts text to uppercase
  • text-breath adds a little bit of letter-spacing than usual to the text. This is primarily used for certain headings or legal text which is rarely used across the site.

Quick layouts

Using the space-between class, a quick navbar-like layout can be achieved. It takes a number of elements and spaces them out within their container, wrapping onto a second line, if required.
